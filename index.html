EmoStar

"Tracking is common. Insight is rare. EmoStar transforms emotional logs into meaningful AI-driven insights."

Dataset / Input

In EmoStar, the primary input consists of user-generated emotional entries which can be:

  • Text-based emotional logs (journaling sentences)
  • Multimodal inputs: optional photos or keyword tags
  • Time-stamped entries to track emotion over sessions

These inputs serve as raw emotional data which the system processes using NLP techniques for sentiment and emotion detection.

Before analysis, text inputs are cleaned, tokenized, and vectorized to support accurate emotion classification and pattern extraction.

Live Execution / Demo

EmoStar is implemented as a functional prototype available on GitHub. Users can interact with the system by entering emotional logs and observing AI-generated emotional insights and trend visualizations.

The system processes real-time user input and demonstrates:

  • Emotion detection from text input
  • Pattern analysis across time-stamped entries
  • Visual representation of emotional trends
  • AI-generated emotional summaries
